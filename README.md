# نظام إدارة الاتصالات الإدارية المتكامل
## CCMS v2.0 — Corporate Communication Management System

---

## ⚠️ مهم جداً — طريقة التشغيل الصحيحة

```bash
python run.py
```

**لا تشغّل** `main.py` أو أي ملف آخر مباشرة.
ملف `run.py` هو نقطة الدخول الوحيدة.

---

## خطوات التثبيت والتشغيل

### الخطوة 1: تثبيت المتطلبات
```bash
pip install flask werkzeug reportlab openpyxl pillow
```

### الخطوة 2: تشغيل النظام
```bash
python run.py
```

### الخطوة 3: فتح المتصفح
```
http://localhost:5000
```

## بيانات الدخول الافتراضية

| المستخدم | كلمة المرور | الدور |
|---------|------------|-------|
| `admin` | `Admin@2025` | مدير النظام الكامل |
| `pm_manager` | `User@2025` | مدير المشاريع |

## هيكل الملفات

```
corr_pro/
├── run.py          ← نقطة التشغيل الرئيسية ✅
├── main.py         ← منطق التطبيق (لا تشغّله مباشرة)
├── models.py       ← قاعدة البيانات والنماذج
├── helpers.py      ← PDF، Excel، مساعدات
├── qr_simple.py    ← توليد رموز QR
├── requirements.txt
├── templates/      ← واجهات HTML
└── instance/       ← تُنشأ تلقائياً (قاعدة البيانات)
```
