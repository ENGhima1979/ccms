💬 إشعارات واتساب
🆓 إعداد CallMeBot (مرة واحدة لكل موظف)
  1. أضف رقم +34 644 59 78 35 في جهات اتصال الموظف باسم CallMeBot
  2. يرسل الموظف من هاتفه هذه الرسالة بالضبط: I allow callmebot to send me messages
  3. سيرد CallMeBot برسالة تحتوي مفتاح API خاص به
  4. أدخل الرقم + المفتاح في إعدادات المستخدم
📱 إعداد UltraMsg (مرة واحدة)
  1. اذهب لـ ultramsg.com وسجّل حساباً مجانياً
  2. اضغط Create Instance
  3. امسح QR Code من واتساب على هاتفك
  4. انسخ Instance ID و Token من اللوحة
  5. أدخلهما أدناه واضغط اختبار
🔵 إعداد Twilio WhatsApp Sandbox
  1. اذهب لـ twilio.com وسجّل حساباً مجانياً
  2. من لوحة التحكم: Messaging → Try it out → Send a WhatsApp message
  3. أرسل الكود المعطى لرقم Sandbox من هاتفك عبر واتساب
  4. انسخ Account SID و Auth Token من أعلى الصفحة
